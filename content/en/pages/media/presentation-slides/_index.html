How to Use This Tool: A Step-by-Step Guide:)

  • Browse: Each presentation has a title (which can be clicked on in the table of contents) and a description of the presentation contents.
  • Explore Visually: Each presentation is featured with a carousel showcasing each of the slides so you can preview it easily. Use the arrows below the carousel to change the image previewed.
  • Access Full Presentations: To explore further, make a copy or download the presentation; click on the link provided under the presentation description. This link directs you to a Google Slides copy of the presentation that you can then edit and adapt to your local context.
